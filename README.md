# CodeLou_FrontEnd

## Description
```
This is my first project for Code Louisville. It's a simple, static HTML webpage using CSS and JavaScript.

```



## Custom CSS IDs
```
The IDs I created are: #nav, #content, #side-bar, #header, #footer.

1. ID #header: sets the height of the header, sets border, uses a vector file for background img, centers text.
2. ID #nav: centers text, sets background color.
3. ID #side-bar: floats text, sets width, padding, font, font-size and color.
4. ID #content: sets padding, margin and main width.
5. ID #footer: sets footer at bottom, sets font, font size. Sets border, padding, line height, bg color.

```
